# mini-react

A monorepo scaffold for learning how a React-like library could be structured.

## Structure

- `packages/mini-react`: core library package (TypeScript-only for now)
- `apps/playground`: Vite playground app that consumes `mini-react`
- `configs`: shared tooling configs

## Setup

```bash
pnpm install
pnpm dev
```
